# Web Anime

This is a web application made in React + Vite + TypeScript where it consumes an external service to display animes by name.

## Tech Stack

**node:** 18.17.0

**react:** 18.3.1

**typescript:** 5.2.2

**vite:** 5.3.1

## Installation

to install the dependencies

```bash
  npm install
```

to run on localhost

```bash
  npm run dev
```

to run the tests

```bash
  npm test
```
